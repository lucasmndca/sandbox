# /under-eight-hours-app-challenge
A challenge where I was supposed to create a full stack application in less than 8 hours.

## Official time taken: 
`~4 hours`

## Stack:
- /frontend
  - AngularJs `1.8.2`
  - CSS 3
  - HTML 5
- /backend
  - NodeJs `17.0.1`
  - Express `4.17.1`
  - MongoDb
